Goldchase Game

Goldchase game is a multi-player game focusing on distributed computing and inter-process communication.

The game is developed in linux enviournment using C++ programming language. The game uses concept of shared

memory where multiple players share a shared memory(each player being a different process shares a comman gameboard)

The communication between two players was made possible by using concepts of signals, message queues and sockets.

Enviornment: Linux, C++, Shared Memory, Semaphore, Inter-process communication, Sockets, Message Queues, Signals.
